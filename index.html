Grammar Race
Grammar Race is a game that utilyzes js, html, and css. This project pulls sentences from a database and allows the user to submit where the comma should go. The correct comma placement and the information provided about the grammar rules are all programatically generated. The user races against a clock and goes up in levels. As they go up in levels, there is a new set of sentences that the program pulls from with new grammatical rules. Watch the video below to see a project demonstration!
